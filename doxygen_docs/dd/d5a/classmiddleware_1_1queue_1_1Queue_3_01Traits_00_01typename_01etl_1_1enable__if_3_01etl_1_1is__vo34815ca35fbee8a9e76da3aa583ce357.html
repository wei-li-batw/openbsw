Detailed Description

template<typename Traits>
class middleware::queue::Queue< Traits, typename etl::enable_if< etl::is_void< typename Traits::LockStrategy >::value >::type >::Receiver

Nested class to read elements from the queue.

After reading an element, the advance method needs to be called in order to clear the current element in the queue and get the to next element.
