uint16_t uds::NestedDiagRequest::getStoredRequestLength ( ::etl::span< uint8_t const > const &  request) const
protectedvirtual

Get needed length of storage for a given request. The default implementation returns the size of the request. Overload this if storing the request consumes more or less bytes.

Note
While the request is stored within the same message buffer that can be used for the response it is crucial to think about memory consumption of the stored request

Store the request. This will be called only if the return value of startRequest() is greater 0. The default implementation stores the complete request. Overload this method if you need to convert the request or if not all of it needs to be stored.
