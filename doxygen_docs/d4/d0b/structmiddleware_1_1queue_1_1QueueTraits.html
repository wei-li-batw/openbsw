Struct encapsulating features for the queue.

Template Parameters
Typethe object type that the queue will contain.
Countthe number of elements of the queue.
Strategythe object that will be used to lock the mutex (by default is void, meaning no lock mechanism should be used).
TypeOfMutexthe mutex type which according to QueueMutex can only be an integer or a pointer to an integer.
