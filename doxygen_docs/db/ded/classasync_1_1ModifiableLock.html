Detailed Description

A synchronization mechanism that blocks threads from accessing a resource.

The ModifiableLock class ensures mutual exclusion, allowing only one thread to access a protected resource or function at a time. When a thread acquires the lock, any other thread attempting to acquire it is blocked until the lock is released. The lock can be acquired either in the constructor or on demand using the lock function. Similarly, the lock can be released either in the destructor or on demand using the unlock function.
