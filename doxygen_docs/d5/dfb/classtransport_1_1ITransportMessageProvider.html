returns a TransportMessage for a given sourceBusId and targetAddress

Parameters
srcBusIdid of bus message is received from
sourceAddressid of TransportMessage's source
targetAddressid of TransportMessage's target
sizesize of the requested TransportMessage
peekslice to the payload of the underlying data buffer
pTransportMessagea pointer to TransportMessage (0L if no message was available) is written to this pointer.
Returns
  • TPMSG_OK: pTransportMessage has been set
  • TPMSG_INVALID_SRC_ADDRESS: sourceAddress is not allowed from srcBusId
  • TPMSG_INVALID_TGT_ADDRESS: requested targetAddress is invalid
  • TPMSG_NO_MSG_AVAILABLE: all params are valid but all TransportMessages are currently locked
  • TPMSG_NOT_RESPONSIBLE: not the correct provider for this request
Note
The TransportMessage must be released by a call to releaseTransportMessage before it can be returned again by this method.

Implemented in transport::TransportRouterSimple, and transport::AbstractTransportLayer::TransportMessageProvidingListenerHelper.
