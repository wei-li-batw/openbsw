Detailed Description

This class implements the UART communication for the Posix platform. It uses the terminal stdout and stdin interfaces for reading and writing data. Initialize the stdout/stdin communication, and retrieve a singleton instance of the Uart class.

This class implements the UART communication for S32K1xx platforms. It follows the method signatures defined in the UartConcept.h file.
