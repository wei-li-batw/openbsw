Detailed Description

template<size_t N>
class io::SplitWriter< N >

Class for writing at once to multiple queues

This "splits" the stream of data into multiple copies, by saving the data to the first queue and then writing it to all of them at commit time.

[TPARAMS_BEGIN]

Template Parameters
NNumber of IWriter this SplitWriter wraps. [TPARAMS_END]
