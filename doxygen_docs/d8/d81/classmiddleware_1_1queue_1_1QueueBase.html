Detailed Description

Base class for a multi-producer single-consumer queue, that is based on a circular buffer implementation.

This class contains a sent_ and received_ attributes which represent writing and reading cursors, which will always be in the range [0, 2*maxSize[, where maxSize is an attribute that is initialized by the init method, that represents the queue's maximum number of elements. This way there are two distinct constellations where sent_ and received_ point to the same element: 1) sent_ == received_ 2) sent_ == (received_ + MAX_SIZE) % (2*MAX_SIZE). Using this trick the ambiguity between the empty and full cases of the queue is resolved without the need for an unused element in the queue: Case 1) means "empty" and case 2) means "full".
