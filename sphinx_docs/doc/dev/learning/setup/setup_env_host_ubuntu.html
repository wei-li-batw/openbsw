Set up environment on Ubuntu 22.04

As it would be impossible to provide setup instructions for each and every flavour of linux available, instead an example setup on just one popular platform - Ubuntu 22.04- is described. The set up will allow you to build images for two target platforms, POSIX and S32K148.

POSIX platform

The simplest target platform to start with is the POSIX platform, which you should be able to build and run on a PC/laptop. ie. You don’t need an embedded platform to get started.

  • Set up build environment for POSIX platform on Ubuntu 22.04 or Windows

S32K148 platform

If you have access to a S32K148 development board then you can build the image for that target, flash it onto your board and test it.

  • Set up build for S32K148 platform on Ubuntu 22.04
  • Flash, debug and test serial console on Ubuntu 22.04
  • Using GDB with the S32K148EVB board via USB
